REST API сервер приложения Todos
1) Открыть проект в Intellij IDEA (начнется автоматическая загрузка (индексация) ресурсов и пакетов, а так же сборка проекта)
2) Создать на сервере СУБД базу с названием "todo-rest-api" (utf8-general_ci)
3) В созданную БД необходимо импортировать из папки 'database' один из sql-скриптов
    (todo-rest-api_only_structure.sql - содержит только структуру таблиц,
    todo-rest-api_with_data.sql - содержит струткуру и данные(пользователь с админ правами tanya@gmail.com, пороль - qwerty))
4) В файле application.properties заменить логин и пароль для доступа к серверу СУБД на свои
5) Запустить проект
